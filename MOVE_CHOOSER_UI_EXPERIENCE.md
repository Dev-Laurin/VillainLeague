## Key User Experience Points

### Visual Clarity
- **Clear affordability**: Brown = can use, Gray = can't use
- **Immediate feedback**: Hover shows details instantly
- **Icons for effects**: Quick visual recognition
- **Consistent theme**: Medieval fantasy throughout

### Intuitive Navigation
- **Scroll to explore**: Easy to see all moves
- **Hover to learn**: Details without committing
- **Click to select**: Simple, one-click selection
- **Auto-hide**: UI disappears when done

### Information Hierarchy
1. **Move name** (largest, gold) - What is it?
2. **Cost** (medium, right side) - Can I afford it?
3. **Description** (hover) - What does it do?
4. **Effects** (hover) - Exact numbers and mechanics
